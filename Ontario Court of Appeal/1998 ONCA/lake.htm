                      ENDORSEMENT
                                
[1]  It was common ground between counsel that the value of the
assets in issue in each of these appeals was greater than $2,000,
and the trustee therefore had an interest in asserting the
position that it did.


[2]  Leave to appeal should have been sought in respect of each
of these appeals. It was not. These were, however, cases in which
leave would have been granted, had it been sought before a single
judge of this court, and leave to appeal is granted in all three
appeals.


[3]  Counsel for the trustee conceded, in our view correctly,
that no higher standard  should apply to the sufficiency of the
description of consumer goods in a security agreement than to the
sufficiency of the description of other types of collateral. In
our assessment, the description of the collateral contained in
the schedule to the security agreement in each of these three
appeals was sufficient, within the contemplation of s. 11(2)(a)
of the Personal Property Security Act, to enable the collateral
to be identified in each of the respective cases.


[4]  The appeals are allowed with costs.
